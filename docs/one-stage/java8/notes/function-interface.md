# Java8 四大函数式接口## `Consumer&lt;T&gt;`：消费型接口```java// 有参数，无返回值void accept(T t)```来看一个简单得例子：```java/** * 消费型接口: Consumer&lt;T> */@Testpublic void test1 () {              consumo(500, (x) -> System.out.println(x)); }public void consumo (double money, Consumer&lt;Double> c) {     c.accept(money);      }```以上为消费型接口，有参数，无返回值类型的接口。## `Supplier&lt;T&gt;`：供给型接口```java// 无参数，有返回值T get()```来看一个简单得例子：```java/** * 供给型接口，Supplier&lt;T> */@Testpublic void test2 () {     Random ran = new Random();    List&lt;Integer> list = supplier(10, () -> ran.nextInt(10));        for (Integer i : list) {         System.out.println(i);    } }      /** * 随机产生 sum 个数量得集合  * @param sum 集合内元素个数 * @param sup  */public List&lt;Integer> supplier(int sum, Supplier&lt;Integer> sup){     List&lt;Integer> list = new ArrayList&lt;Integer>();     for (int i = 0; i &lt; sum; i++) {          list.add(sup.get());     }    return list; }```上面就是一个供给类型得接口，只有产出，没人输入，就是只有返回值，没有入参## `Function&lt;T, R&gt;`：函数型接口```java// 有参数，有返回值R apply(T t)```来看一个简单得例子：```java/** * 函数型接口：Function&lt;R, T> */@Testpublic void test3 () {     String s = strOperar(" asdf ", x -> x.substring(0, 2));    System.out.println(s);    String s1 = strOperar(" asdf ", x -> x.trim());    System.out.println(s1); } /** * 字符串操作  * @param str 需要处理得字符串  * @param fun Function接口  * @return 处理之后得字符传  */public String strOperar(String str, Function&lt;String, String> fun) {     return fun.apply(str);}```上面就是一个函数型接口，输入一个类型得参数，输出一个类型得参数，当然两种类型可以一致。## `Predicate&lt;T&gt;`：断言型接口```java// 有参数，有返回值，且返回值是 boolean 值：boolean test(T t)```下面看一个简单得例子：```java/** * 断言型接口：Predicate&lt;T> */@Testpublic void test4 () {     List&lt;Integer> l = new ArrayList&lt;>();    l.add(102);    l.add(172);    l.add(13);    l.add(82);     l.add(109);    List&lt;Integer> list = filterInt(l, x -> (x > 100));    for (Integer integer : list) {        System.out.println(integer);    }}     /** * 过滤集合  * @param list  * @param pre  * @return */public List&lt;Integer> filterInt(List&lt;Integer> list, Predicate&lt;Integer> pre){     List&lt;Integer> l = new ArrayList&lt;>();    for (Integer integer : list) {        if (pre.test(integer)) {            l.add(integer);         }    }    return l;}```上面就是一个断言型接口，输入一个参数，输出一个 boolean 类型得返回值。
