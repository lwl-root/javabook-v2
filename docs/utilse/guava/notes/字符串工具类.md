# 字符串工具类

对字符串的操作常见的有：

|  #   | 常见操作         | 工具类      |
| :--: | :--------------- | :---------- |
|  1   | 字符串的判断空   | Strings     |
|  2   | 字符串的拼接     | Joiner      |
|  3   | 字符串的拆分     | Splitter    |
|  4   | 字符串内容的替换 | CharMatcher |

## 对字符串的判断空：Strings

中文语境中的『**空**』在编程领域中细分为三种：`null`、`""` 和 `仅含空白符的字符串`。即，英语中的 null、empty 和 blank 。

guava 在 com.google.common.base 包下提供的 **Strings** 工具类中有一个 **isNullOrEmpty** 方法，它的判断逻辑显而易见：

伪代码如下：

```java
Strings.isNullOrEmpty(null)   = true
Strings.isNullOrEmpty("")     = true
Strings.isNullOrEmpty(" ")    = false
```

## 对字符串作拼接：Joiner

guava 在 com.google.common.base 包下为我们提供了 **Joiner** 类来做字符串的拼接、合并。

伪代码如下：

```java
// 结果："hello world"
Joiner.on(" ").join(["hello", "world"]) 

// 结果：sb = "hello, world, goodbye"
StringBuffer sb = new StringBuffer("");
Joiner.on(", ").appendTo(sb, {"hello", "world", "goodbye"})
```

## 对字符串的拆分：Splitter

guava 在 com.google.common.base 包下提供了 **Splitter** 类来做字符串的拆分。

伪代码如下：

```java
// 拆出 4 个："hello", " world", " ", " goodbye"  
Splitter.on(",").split("hello, world, , goodbye");
```

- **.on** 方法用于指明分隔符；
- **.split** 方法表示对目标字符串进行拆分。*split* 方法还有一个 **.splitToList** 的“兄弟”方法 。

由于目标字符串的原因，上述代码拆分的结果中有两样“讨厌”的东西：『空格』和『空字符串』。对此我们可以进一步改进：

```java
// 拆出 3 个："hello", "world", "goodbye"  
Splitter.on(",")
    .omitEmptyStrings()
    .trimResults()
    .split("hello, world, , goodbye");
```

多出来的两个方法调用：

- **.omitEmptyStrings** 表示忽略拆出来的空字符串
- **.trimResults** 方法表示要对每个拆分结果做 **trim** 操作

*.on* 方法有一个更高级的支持正则表达式的“兄弟”方法：**.onPattern** ：

伪代码如下：

```java
// 拆出 4 个："hello" "word" "世界" "和平"
Splitter.onPattern("[,，]{1,}")
    .trimResults()
    .omitEmptyStrings()
    .splitToList("hello,world,,世界, 和平");
```

## 字符串内容的替换：CharMatcher

guava 在 `com.google.common.base` 包下提供了 **CharMatcher** 类用于查找并处理字符串内容。

*CharMatcher* 提供了多种对字符串处理的方法, 它的主要意图有:

1. 找到匹配的字符
2. 处理匹配的字符

通过 *CharMatcher* 对字符串进行操作（包括但不仅包括『替换』），也是分为两个步：选中、匹配字符串中的某部分，而后进行某种操作。

伪代码：

```java
// 结果："hell wrld"
CharMatcher.is('o').removeFrom("hello world")
```

- **is** 方法表示所需要选中、匹配的内容
- **removeFrom** 表示对目标字符串中所选中的内容进行的操作。上例中是删除。

伪代码：

```java
// 结果："heLLo worLd" 
CharMatcher.is('l').replaceFrom("hello world", 'L')
```

CharMatcher 提供的选中、匹配方法包括但不仅包括 **is** 方法，提供的操作方法包括但不仅包括 **removeFrom** 方法和 **replaceFrom** 方法。

更多关于 CharMatcher 对字符串的匹配功能，见另一篇笔记。
