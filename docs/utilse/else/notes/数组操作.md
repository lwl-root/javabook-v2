# 数组相关操作
下面的方法极个别来自于Java JDK在**java.util**包下提供的**Arrays**工具类，其他来自于Spring框架在**org.springframework.util**包下提供了一个**ObjectUtils**工具类。

## 1. 数组判空

::: details 是否是数组：isArray 方法

isArray 方法来源于 Spring 框架的 ObjectUtils 工具类。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.isArray;

boolean isArray(Object obj)
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
isArray(null)    = false
isArray([])      = true
isArray([1,2,3]) = true
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::



::: details 数组判空：isEmpty 方法
isEmpty 方法来源于 Spring 框架的 ObjectUtils 工具类。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.isEmpty;

boolean isEmpty(Object array)
boolean isEmpty(Object[] array)
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
isEmpty(null)    = true
isEmpty([])      = true 
isEmpty([1,2,3]) = false
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;

:::

## 2. 数组的内容查询
::: details 二分查找：binarySearch 方法
binarySearch 方法来自于 java.util.Arrays 工具类。

使用它有前提：要求数组是有序的。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.binarySearch;

int binarySearch(int[] a, int key)
int binarySearch(long[] a, int key)
...
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
binarySearch([0,1,2,3,4,5], 3)  = 3
binarySearch([5,4,3,2,1,0], 3)  = 2
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
binarySearch 方法的返回值是元素在数组中的索引值。
:::

::: details 是否包含 xxx：containsElement 方法

containsElement 方法来源于 Spring 框架的 ObjectUtils 工具类。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.containsElement;

boolean containsElement(Object[] array, Object element)
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
// 它只能用于对象数组，而不能用于基本类型数组。
containsElement([1,2,3], 1)   = true
containsElement([1,2,3], 4)   = false
containsElement([1,2,3], null)= false
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

## 3. 填充与拷贝
::: details 添加：addObjectToArray 方法

addObjectToArray 方法来自于 Spring 框架的 ObjectUtils 工具类。

向数组的末尾 “追加” 新元素。这里的 “追加” 是假追加，原数组并没有发生变化（也不可能发生变化），这里返回的是一个新数组。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.addObjectToArray;

// 向参数数组的末尾追加新元素，并返回一个新数组。
<A, O extends A> A[] addObjectToArray(A[] array, O obj)
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details 填充（赋值）：fill 方法
fill 方法来源于 JDK 自带的 Arrays 工具类。

fill 方法改变的是原数组。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.fill;

// fromIndex 和 toIndex 是一个左闭右开区间：[fromIndex, toIndex) 。
void fill(int[] a, int val) 
void fill(int[] a, int fromIndex, int toIndex, int val);
...

void fill(long[] a, long val) 
void fill(long[] a, int fromIndex, int toIndex, long val);
...
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
fill([1,2,3], 4)       = [4, 4, 4]
fill([1,2,3], 1, 2, 4) = [1, 4, 3]
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details 拷贝：copyOf / copyOfRange 方法
copyOf / copyOfRange 方法来源于 JDK 自带的 Arrays 工具类。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.copyOf;

// from 和 to 是一个左闭右开的区间：[from, to) 
 int[] copyOf(int[] original, int newLength)
long[] copyOf(long[] original, int newLength)
...

 int[] copyOfRange(int[] original, int from, int to)
long[] copyOfRange(long[] original, int from, int to)
...
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
Arrays.copyOf([0,1,2,3,4], 2)) = [0, 1]

Arrays.copyOfRange([0,1,2,3,4], 2, 4)) = [2, 3]
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

## 4. 数组转换
::: details 转成 List：asList / arrayToList 方法
asList 方法来源于 JDK 自带的 Arrays 工具类。

由给定的数组 a ，返回一个固定大小的 List 对象。

asList 返回的 ArrayList 并不是 java.util.ArrayList ，而是 Arrays 的内部类 ArrayList 。

它不支持 add 和 remove 的操作，是只读的。另外，当改变原数组中的元素时，这个 ArrayList 会 “自动” 随之变化。

逻辑上，这里返回的 ArrayList 对象是数组的一个 “视图” 。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.asList;

<T> List<T> asList(T... a);
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
asList([1,2,3]) = [1,2,3]
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details 转成 String：toString / deepToString 方法
toString / deepToString 方法来自于 JDK 自带的 Arrays 工具类。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.toString;

String toString(int[] a)
String toString(long[] a)
...

// 可用于二维数组
String deepToString(int[] a);
String deepToString(long[] a);
...
```
&lt;/CodeGroupItem&gt;
&lt;CodeGroupItem title="伪代码实例"&gt;

```java
toString(null)    = "null"
toString([1])     = "[1]"
toString([1,2,3]) = "[1, 2, 3]"

deepToString(null)          = "null"
deepToString([[1,2],[3,4]]) = "[[1, 2], [3, 4]]"
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details 基本类型数组转包装类数组：toObjectArray 方法

&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.ObjectUtils.toObjectArray;

// 原生基础类型数组 --> 包装类数组
Object[] toObjectArray(Object source)
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::
