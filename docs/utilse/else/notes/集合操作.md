# 集合相关操作
以下工具方法一小部分来自于 Java JDK 在 **java.util** 包下提供的名为 **Collections** 工具类，其它大部分来自于 Spring 框架在 **org.springframework.util** 包下提供的名为 **CollectionsUtils** 的工具类。

## 1. 集合判空

::: details 是否为空：isEmpty 方法
isEmpty 方法来自于 Spring 框架的 CollectionUtils 工具类。

用于判断 List / Set / Map 是否为空。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.CollectionUtils.isEmpty;

boolean isEmpty(Map<?,?> map)
boolean isEmpty(Collection<?> collection)
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

## 2. 集合元素的查询、摘出与统计
::: details 是否包含：contains / containsInstance / containsAny 方法
contains / containsInstance / containsAny 方法来自于 Spring 框架的 CollectionUtils 工具类。

它们都是用来判断集合中是否包含某个对象，它们的区别在于：

- contains 是以迭代器方式进行判断
- containsInstance 是以常规方式进行判断
- containsAny 是判断集合中否包含某些对象中的任意一个

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.CollectionUtils.contains;
import static org.springframework.util.CollectionUtils.containsInstance;
import static org.springframework.util.CollectionUtils.containsAny;

boolean contains(Iterator<?> iterator, Object element)
boolean containsInstance(Collection<?> collection, Object element)
boolean containsAny(Collection<?> source, Collection<?> candidates)
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

binarySearch 方法来自于 JDK 自带的 Collections 工具类。

binarySearch 方法的使用有前提：List 必须是有序的。

binarySearch 方法返回的是元素在 List 中的索引位置。

&lt;CodeGroupItem title="方法声明"&gt;

```java

```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;

## 3. 添加元素
::: details Collection 添加元素：addAll / mergeArrayIntoCollection 方法
**addAll** 方法来自于 JDK 的 java.util.Collections 工具类。

addAll(Collection&lt;? super T&gt; c, T... elements)，往集合中添加元素

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static java.util.Collections.addAll;

<T> boolean addAll(Collection<? super T> c, T... elements)
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details Collection 添加元素：mergeArrayIntoCollection 方法
**mergeArrayIntoCollection** 方法来自于 Spring 框架的 CollectionUtils 工具类。

它实现了和 Collections.addAll 方法同样的功能。它会将参数 array 中的元素添加到参数 collection 中

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.CollectionUtils.mergeArrayIntoCollection;

<E> void mergeArrayIntoCollection(Object array, Collection<E> collection)  
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details Map 添加元素：mergePropertiesIntoMap 方法
mergePropertiesIntoMap 方法来自于 Spring 框架的 CollectionUtils 工具类。

## 4. 集合变序
“变序” 包括：交换元素、排序、反转顺序、“洗牌”

::: details 交换元素位置：swap 方法
swap 方法来自 JDK 的 Collections 工具类。

swap 方法能交换 List 中指定 2 个位置的元素。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static java.util.Collections.swap;

void swap(List<?> list, int i, int j) 
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::

::: details 排序：sort 方法
sort 方法来源于 JDK 的 java.util.Collections 工具类。功能是对参数 list 进行排序。

sort 方法排完序之后 list 的顺序是自然顺序。

&lt;CodeGroup&gt;
&lt;CodeGroupItem title="方法声明"&gt;

```java
import static org.springframework.util.CollectionUtils.sort;

<T extends Comparable<? super T>> void sort(List<T> list)

// 同上，可自定义比较器
<T> 
void sort(List<T> list, Comparator<? super T> c)
```
&lt;/CodeGroupItem&gt;
&lt;/CodeGroup&gt;
:::
