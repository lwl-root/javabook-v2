# 字符串相关操作
下面的方法除了 1 个来自于 Java JDK 在 java.util 包下提供的 Objects 工具类，其它都来自于 Spring 框架在 **org.springframework.util** 包下提供了一个 **StringUtils** 工具类。

## 1. 字符串判空
::: details 是否为空（null）：isNull 方法
<Badge type="warning" text="注意" vertical="middle" /> isNull 方法来自于 JDK 自带的 <b>Objects</b> 工具类，而不是 Spring 的 StringUtils 。

:::: code-group 
::: code-group-item 声明
```java
import static java.util.Objects.isNull;

boolean isNull(Object obj)
```
另外，下面的 isEmpty / hashLength / hashText 的判断规则中都涵盖了 is-null 的判断，无需再多加一个 is-null 的判断。
:::


::: details 是否为空（empty）：isEmpty / hasLength 方法
isEmpty / hasLength 方法都来自 Spring 框架的 StringUtils 工具类。

isEmpty 和 hasLength 方法都能实现 empty 的判断。只要字符串是 **非空** 和 **非空串**，
哪怕其内容无意义（<small>尽是些空格和 tab</small>）， isEmpty 都会返回 false；hashLength 都会返回 true 。

<CodeGroup>
<CodeGroupItem title="声明">

```java
import static org.springframework.util.StringUtils.isEmpty;
import static org.springframework.util.StringUtils.hasLength;

// null 和 empty-string 会返回 true；否则返回 false 。
boolean isEmpty(Object str)

// null 和 empty-string 和 blank-string 会返回 true；否则返回 false 。
boolean hasLength(CharSequence str)
```
</CodeGroupItem>
<CodeGroupItem title="伪代码实例">

```java
isEmpty(null)   = true
isEmpty("")     = true
isEmpty(" ")    = false
isEmpty("xxx")  = false

hasLength(null)   = false
hasLength("")     = false
hasLength(" ")    = true
hasLength("xxx")  = true
```
</CodeGroupItem>
</CodeGroup>

:::

hasText 方法的 “预期” 字符串中有实际的、有意义的内容，这种情况下它会返回 true 。

<CodeGroup>

<CodeGroupItem title="伪代码实例">

hasText("")     = false

</CodeGroupItem>
</CodeGroup>

::: details 判断字符串以 xxx 开始 / 结束 / 包含：PatternMatchUtils 类
PatternMatchUtils 类来自 org.springframework.util 包 。

它用于进行简单地正则匹配。判断规则有：`xxx*` 、`*xxx` 、`*xxx*` 和 `xxx*yyy` 。`*` 通配任意个字符。例如：`hello*` 、`*hello` 、`*hello*` 和 `hello*world` 。

<Badge type="warning" text="注意" vertical="middle" />下述方法的参数中，正则规则字符串在前。

<CodeGroup>
<CodeGroupItem title="声明">

```java
import static org.springframework.util.PatternMatchUtils.simpleMatch;

// 判断字符串是否符合规则。
boolean simpleMatch(String pattern, String str) 

// 判断字符串是否同时满足多个规则。
boolean simpleMatch(String[] patterns, String str) 
```
</CodeGroupItem>
<CodeGroupItem title="伪代码实例">

```java
PatternMatchUtils.simpleMatch("hello*", "hello world");      = true
PatternMatchUtils.simpleMatch("*hello", "hello world");      = false
PatternMatchUtils.simpleMatch("*hello*", "hello world");     = true
PatternMatchUtils.simpleMatch("hello*world", "hello world"); = true
```
</CodeGroupItem>
</CodeGroup>
:::

```java

import static org.springframework.util.StringUtils.mergeStringArrays;

String[] concatenateStringArrays(String[] array1, String[] array2) 

// 被废弃，建议通过 LinkedHashSet 手动合并两个字符串。

```
</CodeGroupItem>
